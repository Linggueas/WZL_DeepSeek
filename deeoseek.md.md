# &#x20;DeepSeek AI 智能对话客户端

桌面端AI对话工具，支持多会话、深度思考模式、流式实时输出，对话自动本地存储，回
答可导出txt。

## 功能特性

*   1.该项目基于 DeepSeek API 构建，实现了流畅的 AI 对话功能。
*   2.在架构上，采用 Qt MVC 模式进行设计，仅维护一个统一的视图（View），并为每个对话会话独立分配数据模型（Model），从而确保了界面响应的高性能与良好的代码组织。
*   3.为保障对话之间的数据隔离，每个对话会话使用独立的 QJsonArray 存储历史记录，有效避免了上下文混淆与数据污染。
*   4.此外，项目集成了本地 SQLite 数据库，支持对话历史的持久化保存，便于用户随时回溯与管理。
